Echo用户帮助手册

一、引言

👋 欢迎使用Echo-SIT团队开发的校园小程序

本小程序是一款面向高校师生的多功能校园服务平台，基于 TypeScript、Vue 3 和现代前端技术栈开发。该小程序支持课表、成绩、消费、图书馆、校园地图等多种实用功能，致力于提升校园数字化体验。

二、如何开始使用

1.打开微信：在手机主屏幕找到并点击微信图标。

2.搜索小程序：

  • 方法一：在微信首页向下滑动，在顶部搜索框中输入“Echo回应”

  • 方法二：点击微信右上角的“搜索”图标，选择“小程序”，然后输入名称“Echo回应”进行搜索。

3.首次使用授权：

  • 找到对应的小程序后，点击进入。

  • 首次使用时，系统会提示您输入您的学号/工号及对应密码（就是登录学校OA的密码），以验证您的在校身份。

三、主要功能和使用指南

3.1 课表查询

  • 入口：启动小程序后，默认显示即为课表页面。
  • 课表导入：首次使用或学期更新时，请点击页面左上角的设置按钮，依序选择“更改课表”即可完成导入。若遇学校临时调休，请在此处选择对应的“调休课程”以获取准确安排。
  • 查看与切换：
    • 切换周次：在课表页面左右滑动屏幕，即可灵活切换查看上一周或下一周的课程安排。
    • 课程详情：点击任一课程卡片，会弹出窗口展示该课程的详细信息（如任课教师、具体上课地点等）。
    • 返回本周：点击右上角的 “返回本周” 按钮，可一键快速定位回当前周的课表。
  • 课表导出/订阅： 您可以将课表导出并订阅到您手机的系统日历中，从而实现课程安排的自动同步与提醒。
    • 点击页面左上角的设置按钮。
    • 选择“导出课表”功能，系统将生成一个标准的 iCalendar (.ics) 订阅链接。
    • 订阅方式：
      1. iOS / Windows 用户：复制此链接后，通常可在Safari或Edge等浏览器中打开，系统会自动引导您将课程表订阅到系统自带的“日历”应用中。
      2. 安卓用户：建议将链接复制后，在您常用的日历应用中，找到“添加其他日历”或“通过链接订阅”功能，粘贴此链接即可完成订阅。

3.2 成绩与绩点查询

  • 入口：点击底部导航栏 「工具」 -> 选择 「成绩查询」。
  • 成绩查询：
    • 页面默认显示当前学年学期的成绩列表。
    • 可通过顶部菜单切换不同的学年和学期，查询历史成绩。
    • 点击具体科目，可查看该课程的详细成绩构成。
  • 绩点计算：可使用内置的绩点计算器，自主选择特定课程进行加权平均分和预估绩点的计算。

3.3 消费查询

  • 入口：在 「工具」 页面选择 「消费查询」。
  • 校园卡服务：页面顶部会清晰显示您的校园卡实时余额。
  • 电费查询：
    • 在指定输入框中，准确填写您所在的楼号和寝室号。
    • 点击 「查询」 按钮，即可获取当前剩余电费信息。
    • 系统支持切换查看 24小时 或 12小时 内的电费消耗趋势图，助您掌握用电规律。

3.4 图书馆服务

  • 入口：在 「工具」 页面选择 「图书馆」。
  • 馆藏检索：
    • 在搜索框中输入关键词，即可快速查找您所需的图书资料，并实时查看其在馆状态、具体架位等信息。
    • 您可以通过搜索框左侧的筛选按钮，灵活调整搜书方式，支持按标题、作者、主题、出版社、ISBN、索书号、分类号等多种维度进行精准检索。
    • 提供 “本月热搜榜” 与 “热搜总榜” ，助您了解热门读物，发现阅读兴趣。

3.5 校园地图

  • 入口：在 「工具」 页面选择 「校园地图」。
  • 功能：可浏览校园整体建筑布局，点击具体建筑图标即可查看其名称、功能介绍等详细信息，是您校园导览的得力助手。

3.6 校园电话黄页

  • 入口：在 「工具」 页面选择 「校园电话」。
  • 功能：查看学校各部门、常用服务机构的联系电话列表。点击任一电话号码，即可直接发起呼叫。

3.7 考试安排

  • 入口：在 「工具」 页面选择 「考试安排」。
  • 功能：集中查看当前学期的所有考试信息，包括考试科目、具体时间、考场地点等关键详情，助您高效备考。

3.8 第二课堂

  • 入口：在 「工具」 页面选择 「第二课堂」。
  • 功能：
    • 查看个人已获得的第二课堂学分情况及各类活动的学分要求。
    • 点击 「查看进行中活动」 ，了解当前可报名的活动列表。
    • 点击 「查看全部活动」 ，浏览所有历史与未来活动。

3.9 公告与新闻

  • 入口：在 「工具」 页面可直接浏览最新公告列表。
  • 功能：点击具体公告标题即可查看全文。此外，可通过 「新闻详情」 页面获取更多校园资讯和新闻报道。

3.10 校历工具

  • 入口：在 「工具」 页面选择 「校历」。
  • 功能：直观展示本学期校历，帮助您快速了解教学周次、重要节假日、寒暑假安排等关键日期。

3.11 自定义设置

  • 入口：在课表页面点击左上角的设置按钮，选择「自定义设置」。
  • 更换背景：
    • 点击「更换背景」，可从手机相册选择图片或使用系统提供的默认背景图，自定义课表页面背景。
  • 切换主题：
    • 点击「切换主题」，可选择多种预设主题，包括：
      • 默认：保留系统原始配色与布局
      • 简约：采用简洁线条与纯色块，减少视觉干扰
      • 温暖：使用暖色调（如浅橙、米黄），营造温馨氛围
      • 梦幻：采用渐变色彩与柔和元素，增添梦幻感
      • 冷色：以蓝、灰色调为主，呈现冷静、专业风格
    • 切换后，页面配色、图标风格等将实时更新，提升个性化体验。

四、注意事项

为确保您能顺利使用本小程序，请留意以下事项：

  1. 登录验证：部分涉及个人数据的功能（如成绩、课表、消费查询）需要您完成登录授权后方可使用。
  2. 数据同步：小程序内大部分数据与学校官方系统同步，可能存在一定延迟，请以学校各主管部门发布的最终数据为准。
  3. 测试功能提示：“第二课堂”功能目前处于测试阶段，相关数据可能存在延迟或不准确的情况，仅供参考。
  4. 异常处理：如遇功能异常或页面加载失败，建议尝试退出小程序后重新进入，或检查设备网络连接是否正常。
  5. 使用规范：为避免对服务器造成不必要的压力，请勿在短时间内过于频繁地刷新数据。

五、常见问题解答 (FAQ)

Q1: 课表无法显示或显示空白怎么办？

  • A1：请按以下步骤排查：
    • 确认您已通过左上角设置按钮成功导入课表。
    • 检查手机网络连接是否稳定。
    • 尝试退出账号并重新登录。

Q2: 成绩查询为何没有数据？

  • A2：可能原因有：
    • 当前所选的学年或学期确实尚无成绩记录。
    • 您尚未完成该学期课程的教师评价等必要前置步骤。

Q3: 电费查询失败是什么原因？

  • A3：请检查：
    • 输入的楼号和寝室号格式与内容是否完全正确。
    • 网络连接是否通畅。
    • 系统可能处于临时维护时段，请稍后再试。

Q4: 遇到功能异常或页面空白如何解决？

  • A4：通用解决方案：
    • 完全关闭微信后台的小程序，然后重新打开。
    • 检查微信客户端是否为最新版本。
    • 确认当前使用的网络信号良好。

七、附录

用户协议及隐私政策

本小程序名为"Echo回应"，由上海应用技术大学回应工作室团队（EchoTech Studio，大学生大创实践项目）负责后续运营与维护，所有成员均为本校学生。Echo回应的服务主要基于上海应用技术大学教务系统及相关校内服务，由本团队负责技术支持与日常运维。使用本软件即代表您理解并同意以下条款：

  1. 数据代理说明

为提供稳定的服务，Echo回应将通过自有的代理服务器中转您与上海应用技术大学校园网服务（通常位于 .sit.edu.cn 域名下）之间的网络请求。这意味着您在使用本小程序时，相关请求和数据将经由我们的服务器进行转发。

我们郑重承诺：在此中转过程中，您的个人信息（包括但不限于学号、密码、成绩、课表等）不会被我们的服务器存储或记录。代理服务器仅做实时转发，不会留存任何用户数据。

  1. 校园网服务说明

校园网服务由上海应用技术大学官方提供并运营，Echo回应不对其服务的可用性、稳定性或数据的准确性作任何担保。如在使用过程中遇到校园网服务相关问题，请及时联系学校相关部门处理。

您授权Echo回应以用户代理（User Agent）身份，使用您的身份信息访问校园网站点，以便为您提供相关服务。在此过程中，您的账号及身份信息仅用于您所请求的校园网服务，不会被我们保留或用于其他目的。

  1. 数据安全与隐私保护

我们采取业界流行的加密传输等安全技术措施，尽力保护您的数据在传输过程中的安全。您的所有个人信息仅用于实现您所请求的服务功能，Echo回应不会向任何第三方提供、出售、出租、分享或交易您的个人信息。

  1. 本地数据存储说明

为提升用户体验，Echo回应会在您的设备本地存储以下数据：

  • 登录凭据（学号、密码加密形式）：用于维持登录状态
  • 课程表、成绩等查询结果缓存：减少网络请求，提升加载速度
  • 用户个性化设置：如用户头像、界面主题、偏好设置等

所有本地存储数据均采用设备原生加密机制保护，且您随时可以通过清除小程序缓存来删除这些数据。

  1. 数据缓存策略

为提高服务响应速度，Echo回应对以下数据采取本地缓存策略：

  • 课程表、成绩等数据
  • 用户基本信息
  • 静态数据（如校园电话等）

缓存数据用于改善用户体验，对数据的实时性有一定的影响，望您知悉。您可随时通过“刷新缓存”功能来获取最新数据。

  1. 用户权利

您享有以下权利：

  • 知情权：了解个人信息的处理方式和目的
  • 访问权：随时查看小程序存储的个人信息
  • 删除权：通过清除小程序缓存删除所有本地数据
  • 撤回同意权：停止使用即视为撤回数据处理同意
  1. 用户义务

您在使用Echo回应时，应遵守国家法律法规、学校规章制度和社会公序良俗，不得利用本程序进行任何违法、违规或损害他人利益的行为。

Echo回应团队有权对任何异常使用行为进行调查，并保留暂停服务、记录相关信息及移交学校有关部门处理的权利。

  1. 安全事件响应

如发生个人信息泄露等安全事件，我们将：

  1. 及时通过适当方式告知您

  2. 按照法律法规要求上报主管部门

  3. 采取合理有效措施减少损失

  4. 进行安全漏洞排查和修复

  5. 协议变更说明

如本协议发生更新，我们将通过适当方式提醒您。如您不同意变更后的协议，可联系我们删除与您账户相关的信息。

  • 微信公众号：SIT回应工作室
